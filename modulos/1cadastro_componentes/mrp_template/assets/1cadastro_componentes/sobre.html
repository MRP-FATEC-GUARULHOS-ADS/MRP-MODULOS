@Sobre - Saco de Carvão Ltda.

O sistema pelo qual você está navegando no presente momento é um projeto proposto pela Professora Jane Maria, professora de Linguagem de Programação da FATEC Guarulhos. Consiste na criação do MRP de uma empresa montadora de computadores, onde cada um dos 5 grupos da sala ficou responsável por uma parte do sistema, sendo este mui humilde grupo responsável pelo módulo de cadastro de peças para produção.
